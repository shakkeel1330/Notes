About ML²

We are a not-for-profit organization that focuses on addressing machine learning limitations using machine learning. ML² is a space for machine learning enthusiasts to collaborate and find solutions for such limitations. Our current projects, IDLY and IMLY adorns the same philosophy.

Interpretable Deep Learning

One of the most common, if not the most popular, pain points for ML practioners is the interpretabilty of the models they create. With IDLY, we are attempting to find a holistic solution for the interpretability issue faced by ML practitioners. Our approach is to find an almost all encompassing deep learning algorithm as a solution to this problem.

Interoperable Machine Learning

Another pain point widely faced by ML practioners is the lack of a tool that easily lets you share saved model across teams. IMLY is expected to be a tool that would let you save a model in a given framework(say Scikit Learn) and be imported to an entirely different machine learning framework(say Keras). Such a tool would provide flexibility amongst teams that use multiple ML frameworks.
